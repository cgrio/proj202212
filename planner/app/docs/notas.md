## Documentação e notas do aplicativo ##
Comandos úteis (https://docs.flutter.dev/reference/flutter-cli)

Criar aplicativo
flutter create my_app
cd my_app
flutter analyze
flutter test
flutter run lib/main.dart



Relacionados as bibliotecas pub
 flutter pub get
 flutter pub outdated
 flutter pub upgrade

Rodar e testar a aplicação
Listar emuladores
flutter emulators

Iniciar um emulador
flutter emulators --launch New_Device_API_29

rodar os scripts de teste
flutter test [<DIRECTORY|DART_FILE>]
